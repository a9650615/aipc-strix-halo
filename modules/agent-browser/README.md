# agent-browser

Playwright MCP browser integration (D6).

Playwright MCP provides browser automation for the agent framework.
The MCP server runs inside a node distrobox; config declares it as
a registered tool for the agent and dev consumer groups.

## Dependencies
- dev-ai-mcp-dev-servers (playwright entry in servers.json)
- llm-litellm
