# CCS (Claude Code Switch) Module

CCS is a multi-provider profile and runtime manager for Claude Code and compatible CLIs. 

It allows you to switch between multiple runtimes (Claude, Codex, Droid), multiple Claude subscriptions/accounts, and various API providers (OpenRouter, Ollama, etc.) without configuration friction.

## Features
- **Multi-runtime support**: Claude Code, Factory Droid, Codex CLI.
- **Account Management**: Switch between different Claude accounts and OAuth providers.
- **Provider Bridge**: Use OpenAI-compatible routing for Anthropic-compatible proxies.
- **Unified Command Surface**: A single CLI interface to manage complex AI workflows.

## Installation (within OpenCode)
This module is managed via the OpenCode module system. Deployment depends on your target (bootc/ansible).
